    Browser Rendering Pipeline

    HTML

    Browser receives HTML source.

    ↓

    Parsing

    HTML parser processes the document.

    ↓

    Tokenization

    HTML is converted into tokens.

    ↓

    DOM Tree

    Tokens become DOM nodes.

    ↓

    CSS

    Stylesheets are downloaded and parsed.

    ↓

    CSSOM Tree

    CSS rules become the CSS Object Model.

    ↓

    DOM Tree + CSSOM Tree

    Browser combines structure and styles.

    ↓

    Render Tree

    Visible elements are prepared for layout and painting.
