DOM Explorer – Task Manager is a modern, responsive web application developed using HTML5, CSS3, and Vanilla JavaScript (ES6+). The project is designed to demonstrate a strong understanding of core browser concepts and native DOM APIs without using any external frameworks, libraries, UI kits, or dependencies. Its primary objective is to showcase how JavaScript interacts directly with the Document Object Model (DOM) to create dynamic, interactive, and efficient user experiences.

The application enables users to create, edit, complete, delete, search, and filter tasks through an intuitive and responsive interface. Each task is generated dynamically using native DOM methods such as `createElement()`, `createTextNode()`, `append()`, and `appendChild()`. Custom data attributes including `data-id`, `data-status`, and `data-category` are used to demonstrate the practical implementation of `dataset`, `setAttribute()`, `getAttribute()`, `hasAttribute()`, and `removeAttribute()`.

To improve performance and maintain clean code architecture, the project uses event delegation by attaching a single event listener to the parent task container and handling user interactions through `event.target` and `closest()`. The application also demonstrates event propagation concepts, including event bubbling and event capturing, through an interactive visual example.

Additional educational sections explain the difference between HTML attributes and JavaScript properties by comparing `input.value` with `input.getAttribute("value")`. A visual browser rendering pipeline further illustrates the process of HTML parsing, tokenization, DOM Tree creation, CSSOM Tree generation, and Render Tree construction.

The application includes local storage integration to persist task data and theme preferences across browser sessions. With its glassmorphism-inspired design, dark and light theme support, accessibility-focused structure, and modular JavaScript architecture, DOM Explorer – Task Manager serves as both a functional productivity tool and a practical learning project for mastering Vanilla JavaScript and browser internals.
